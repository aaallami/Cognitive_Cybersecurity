# datasets/

- `synthetic_flows.csv` — generated by notebook 01 / `experiments/run_baseline_benchmark.py`
  (schema-compatible synthetic data; see `docs/data_provenance.md`).
- `raw/` — place real dataset files here (e.g. `UNSW_NB15.csv`, `CICIDS2017_Monday.csv`).
  Not committed to git (see `.gitignore`); each user supplies their own copy.
- `processed/` — pickled train/val/test splits produced by notebook 01
  (`splits.joblib`). Regeneratable; not committed to git.
